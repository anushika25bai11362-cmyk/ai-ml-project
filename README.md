##  Description
This project is a Python-based application that identifies a user's hair type by asking a few simple questions about their hair condition. Based on the responses, it classifies hair into oily, dry, normal, or combination and provides suitable haircare tips along with recommended and harmful chemicals.

---

##  Objective
- To determine hair type using user input  
- To provide basic haircare suggestions  
- To recommend chemicals to use and avoid  

---

##  Technologies Used
- Python  
- Conditional statements (if-else)  
- Dictionary (scoring system)  

---

# How It Works
1. The program asks questions related to hair condition  
2. Each answer increases the score of a specific hair type  
3. The hair type with the highest score is selected  
4. Haircare tips and chemical recommendations are displayed  

## Features
- Interactive quiz  
- Instant hair type detection  
- Personalized suggestions  
- Beginner-friendly code
